# README

# Requirement
Docker
docker-compose

# Usage
``` bash
$ git clone git@github.com:gettan1023/frontend.git frontend
$ cd frontend
$ echo "PORT=3010" > .env
$ docker-compose up --build
```

# Open localhost:3010/

# Change remote repository
```
$ git remote set-url origin https://github.com/USERNAME/REPOSITORY.git
```
